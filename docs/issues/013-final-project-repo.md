# 013 — Final Project 独立仓库

## Parent

[PRD: CS50x 2026 个人学习日志仓库](../prd.md)

## What to build

CS50 Final Project 将在 GitHub 上建立独立仓库（不放在课程学习日志仓库内）。该仓库用于展示课程最终的完整软件项目，独立于课程练习。

本 issue 的任务是：创建该独立 GitHub 仓库，包含基础的 `README.md` 和 `.gitignore`，仓库名和项目选题待定（届时由学习者自行决定）。

## Acceptance criteria

- [ ] 独立的 GitHub 仓库已创建（名称待定，如 `cs50-final-project`）
- [ ] 仓库 README 说明这是 CS50x 2026 的最终项目
- [ ] `.gitignore` 根据所选技术栈配置
- [ ] 不与本课程仓库产生文件耦合

## Blocked by

None — 独立仓库，与课程仓库无依赖。可在课程进行到 Final Project 阶段时再创建。
