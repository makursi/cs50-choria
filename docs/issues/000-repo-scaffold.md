## What to build

初始化整个 CS50x 2026 学习日志仓库的基础骨架：

- `git init` 初始化本地 Git 仓库
- 创建极简 `README.md`（仅简要介绍仓库用途）
- 创建 `.gitignore`（忽略 C 编译产物 `*.exe` `*.o` `*.out`、Python 缓存 `__pycache__/`、IDE 配置 `.vscode/` `.idea/`）
- 创建 11 个 `week-N-topic/` 空文件夹作为后续各周学习的容器

每个 week 文件夹内暂不放置任何内容，后续 issue 各自填充。

## Acceptance criteria

- [ ] `git init` 已执行
- [ ] `README.md` 存在，内容极简，仅介绍这是 CS50x 2026 个人学习日志
- [ ] `.gitignore` 存在，覆盖 C 产物、Python 缓存、IDE 配置
- [ ] 以下目录全部存在（空目录，可含 `.gitkeep`）：
