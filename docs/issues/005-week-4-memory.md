## What to build

完成 Week 4（内存）的学习笔记和全部问题集解答。该周主题：指针、动态内存分配（malloc/free）、栈与堆、文件 I/O、图像处理基础。

在 `week-4-memory/` 下创建 `notes.md` 及以下源代码文件，覆盖 Less + More 两个难度：

- `volume.c` — 音频音量调节（Lab）
- `filter-less.c` — 图像滤镜：灰度、深褐色、反射、模糊（Less）
- `filter-more.c` — 图像滤镜：额外包含边缘检测（More）
- `recover.c` — 从磁盘镜像中恢复 JPEG 文件
